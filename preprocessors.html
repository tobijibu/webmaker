Pre-processors

更新日: 2018.11.13

Web MakerはHTML、CSS、JavaScriptのプリプロセッサをサポートしています。 それぞれのペインのタイトルをクリックして、いずれかのプリプロセッサに切り替えることができます。

利用可能なプリプロセッサは以下の通りです。

HTML

  • HTML
  • Markdown
  • Pug

CSS

  • CSS
  • SCSS
  • SASS
  • LESS
  • Stylus
  • Atomic CSS

JavaScript

  • JS
  • CoffeeScript
  • ES6(Babel)
  • TypeScript
